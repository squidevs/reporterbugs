# Work Item Generation Assistant - Project Summary

## Resumo do Projeto (4 linhas)

O **Work Item Generation Assistant** é uma ferramenta web que padroniza a criação de work items através de duas interfaces: **gerar-novo.html** (https://geradorbugs.web.app/gerar-novo.html) para uso interno pela equipe de QA e desenvolvimento na criação de bugs e defeitos, e **auxiliar-de-geracao-de-work-item.html** (https://geradorbugs.web.app/auxiliar-de-geracao-de-work-item.html) para uso externo pelos clientes na criação de bugs e user stories.

O objetivo principal é garantir que todas as tarefas estejam padronizadas e contenham todas as informações essenciais (contexto, passos de reprodução, ambiente, usuário, regras de negócio, features relacionadas, service layer, etc.) para que a equipe possa trabalhar eficientemente sem necessidade de retrabalho ou solicitação de dados adicionais.

A ferramenta valida campos obrigatórios em tempo real, gera textos formatados prontos para cópia em sistemas de gestão de projetos, e integra-se com Slack para notificações automáticas em canais específicos (QA ou CLI conforme o tipo de demanda).

Além disso, oferece funcionalidades de importação/exportação de templates JSON e CSV, organização de features/pages por grupos, e geração de work items padronizados que seguem um formato consistente, facilitando a comunicação e o trabalho da equipe de desenvolvimento.
